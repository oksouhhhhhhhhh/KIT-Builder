# KIT Builder
A GitHub page to create custom KITs for the Roblox randomizer experience "item asylum". Feel free to contribute towards this to make it better. All of the images used in this project are made by [Jean's Bizarre Community](https://www.roblox.com/communities/6356872/Jeans-Bizarre-Community) (https://www.roblox.com/communities/32333184/Hexacrunch).

You can find the link to the GitHub page here: https://oksouhhhhhhhhh.github.io/KIT-Builder/
